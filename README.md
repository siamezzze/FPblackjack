FPblackjack
===========

Blackjack game

 - cabal configure
 - cabal build
, then:
 - cabal run player (optional IP adress) (port number)
for every player (they will then have to wait for dealer to start the game)
 - cabal run dealer
for dealer

For example:
 - cabal run player 4567
 - cabal run player 192.168.0.9 4568
 - cabal run dealer

Enjoy!
